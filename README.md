# نظام سريع (SAREE3) 🛠️
منصة إلكترونية لطلب خدمات الصيانة المنزلية، تخدم مناطق الولي، الحلاوين، والقري المجاورة.

### المميزات:
- **تطوير:** BRAZILI
- **التقنيات:** HTML5, CSS3 (Glassmorphism), JavaScript.
- **الربط:** نظام WhatsApp API لإرسال الطلبات مباشرة.
- **الدفع:** يدعم توضيح وسيلة الدفع (كاش، بنكك، أوكاش).

تاريخ الإطلاق: مارس 2026.
